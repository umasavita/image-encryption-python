# 🖼️ Image Encryption using Python

## 📌 Description
This project encrypts and decrypts images by modifying pixel values using a key.

## 🛠 Tools Used
- Python
- OpenCV

## 🚀 Features
- Encrypt image using key
- Decrypt image back to original
- Simple CLI-based program

## ▶️ How to Run
pip install opencv-python
python image_encrypt.py


## 📸 Output
- encrypted.png
- decrypted.png

## 📚 Learning
- Image processing basics
- Pixel manipulation
- Encryption concept
